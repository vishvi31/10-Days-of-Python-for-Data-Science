## 10-Day Curriculum

| Day | Topic | Level |
|---|---|---|
| Day 01 | Variables, Data Types, Conditionals | Beginner |
| Day 02 | Loops and List Comprehensions | Beginner |
| Day 03 | Functions, Recursion, Decorators | Beginner+ |
| Day 04 | OOP - Classes, Objects, Inheritance | Intermediate |
| Day 05 | NumPy - Arrays and Operations | Intermediate |
| Day 06 | Pandas - DataFrames and EDA | Intermediate |
| Day 07 | Data Visualisation - Matplotlib and Seaborn | Intermediate |
| Day 08 | Machine Learning - Sklearn Pipelines | Advanced |
| Day 09 | NLP - Text Processing and TF-IDF | Advanced |
| Day 10 | Capstone - End-to-End DS Project | Advanced |
